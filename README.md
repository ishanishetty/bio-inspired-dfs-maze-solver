# Bio-Inspired DFS Maze Solver

This project implements a maze solving system based on the Depth-First Search (DFS) algorithm and explores bio-inspired optimization techniques to improve traversal efficiency.

The system compares traditional DFS traversal with optimized approaches using Ant Colony Optimization (ACO) and Artificial Bee Colony (ABC) algorithms.

The goal is to analyze how bio-inspired strategies can improve search performance in maze solving problems.

---

## Features

- Maze solving using Depth-First Search
- Bio-inspired optimization using Ant Colony Optimization
- Artificial Bee Colony based search improvement
- Visualization of maze traversal paths
- Comparison between traditional and optimized DFS
